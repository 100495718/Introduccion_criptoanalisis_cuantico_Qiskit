## Requisitos

El proyecto se ha creado utilizando Python 3.12.9. Para ejecutarlo, se necesitan las siguientes dependencias: 

- Qiskit 2.3.1
- Matplotlib 3.10.8
- Numpy 2.4.1
- Notebook 7.3.2

Pueden instalarse con:

```bash
pip install -r requirements.txt
```
